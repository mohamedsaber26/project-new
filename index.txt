- string concatenation

var firstName = "John";
var lastName = "Doe";
var fullName = firstName + " " + lastName;
console.log(fullName); // Output: "John Doe"


في هذا المثال، تم دمج متغيرين نصيين firstName و lastName باستخدام عامل الإضافة +، وتم إضافة مسافة بينهما. الناتج هو سلسلة نصية تحتوي على اسم كامل "John Doe".

بالإضافة إلى استخدام عامل الإضافة +، في بعض اللغات البرمجية الأخرى قد تكون هناك وسائل أخرى للدمج. على سبيل المثال، في Python، يمكن استخدام عامل الإضافة + بالإضافة إلى عملية الدمج باستخدام str.join()،
 وفي لغة البرمجة Swift يمكن استخدام عامل الإضافة + بالإضافة إلى عملية الدمج باستخدام String interpolation.
